CSS

Cascading Style Sheet (CSS) merupakan standar untuk mengatur tampilan yang ada pada suatu berkas HTML, CSS bukan bahasa pemrograman tetapi merupakan bahasa deklaratif yang dinyatakan langsung untuk menyatkan suatu tampilan, tidak terdapat logika, percabangan, ataupun perulangan.
